How to get Airtel configuration settings: Complete guide 2022

Airtel is among the most popular telecommunication companies in Nigeria today. The network’s subscribers have access to numerous services, the most common being voice calls, messages, and internet browsing. For the latter to work, a person’s device must have the requisite browsing settings installed. This, then, implies that figuring out how to get Airtel configuration settings is quite essential for those subscribing to the network.

The configuration settings required to give a device access to the internet on a mobile network are known as APN (access point names) settings. While some devices will automatically work when a SIM card is inserted, others might have a problem, requiring the user to install the settings manually.

How to get Airtel configuration settings

Here is a look at the different ways one can get and install Airtel configuration settings.

How to get Airtel configuration settings via SMS

This is known as the automatic configuration method. Here is how to get a configuration message in Airtel.

  • On your device, open the messaging app.
  • Enter the recipient as 232
  • Type ‘MO’ or ‘phone type and model’ and send the text. The latter should look like Nokia 2030.
  • You will receive an Airtel configuration message with the APN settings. There will also be an on-screen prompt asking you to save the settings.
  • Press yes or okay.

    • Manual installation
